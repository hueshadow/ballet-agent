## ADDED Requirements

### Requirement: 机构管理
系统 SHALL 支持用户在"我的"入口管理常去的芭蕾机构，包含新建、编辑、删除操作。机构字段：名称（必填）、城市（选填）、地址（选填）、logo（选填，图片上传）。

#### Scenario: 新建机构
- **WHEN** 用户在机构管理页点击新建
- **THEN** 展示表单，填写名称后可保存
- **THEN** 新机构出现在机构列表中

#### Scenario: 编辑机构
- **WHEN** 用户点击已有机构的编辑入口
- **THEN** 展示预填表单，修改后保存成功

#### Scenario: 删除机构
- **WHEN** 用户点击删除某机构
- **THEN** 弹出二次确认弹窗
- **WHEN** 用户确认删除
- **THEN** 机构从列表移除，关联的老师不删除但解除关联

#### Scenario: 新建上课记录时快速创建机构
- **WHEN** 用户在新建上课记录页选择机构时找不到目标机构
- **THEN** 可点击"快速新建"，填写名称后立即创建并选中

### Requirement: 老师管理
系统 SHALL 支持用户管理老师信息，包含新建、编辑、删除操作。老师字段：名称（必填）、所属机构（多选，联动机构列表）、简介（选填）、头像（选填）。

#### Scenario: 新建老师
- **WHEN** 用户在老师管理页点击新建
- **THEN** 展示表单，名称必填，机构可多选
- **THEN** 新老师出现在老师列表中

#### Scenario: 按机构筛选老师
- **WHEN** 用户在新建上课记录页先选择了机构
- **THEN** 老师下拉列表仅显示属于该机构的老师（及"全部"选项）

#### Scenario: 新建上课记录时快速创建老师
- **WHEN** 用户在新建上课记录页选择老师时找不到目标老师
- **THEN** 可点击"快速新建"，填写名称后立即创建并选中，自动关联当前已选机构
