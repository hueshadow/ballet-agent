## ADDED Requirements

### Requirement: 用户档案创建与编辑
系统 SHALL 支持用户创建和编辑个人档案，包含头像、昵称（从微信资料获取）、舞龄（月数）、个人简介（选填）、学习目标（选填）、每周课时目标（默认 3-4 节）。

#### Scenario: 首次进入引导填写
- **WHEN** 用户首次打开小程序且 users 集合中无该用户记录
- **THEN** 展示引导页，引导用户授权微信头像昵称并填写舞龄和学习目标
- **THEN** 保存后跳转首页

#### Scenario: 编辑已有档案
- **WHEN** 用户在"我的"页面点击编辑
- **THEN** 展示预填当前值的表单，修改后保存成功

#### Scenario: 修改周目标
- **WHEN** 用户在"我的"页面修改每周课时目标
- **THEN** 新目标立即生效，首页周进度条同步更新

### Requirement: 课型词典管理
系统 SHALL 预置 8 个课型（基训、把杆、中间、跳跃、足尖、编舞、汇演排练、其他），并支持用户自定义新增课型。自定义课型存入 courseTypes 集合，在新建上课记录时可选。

#### Scenario: 用户新增自定义课型
- **WHEN** 用户在课型管理页输入新课型名称并保存
- **THEN** 新课型出现在课型列表中，后续新建上课记录时可选

#### Scenario: 预置课型不可删除
- **WHEN** 用户尝试删除预置课型
- **THEN** 系统提示"预置课型不可删除"
