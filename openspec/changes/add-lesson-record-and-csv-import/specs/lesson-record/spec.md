## ADDED Requirements

### Requirement: 新增上课记录
系统 SHALL 支持用户手动新增上课记录，包含以下字段：日期（默认今天可改）、机构（从列表选/快速新建）、老师（按机构联动筛选/快速新建）、课程类型（联动课型词典含自定义）、课程级别（初/中/高）、时长（默认 60 分钟可改）、课后照片（不限张数，前端压缩）、课堂笔记（支持语音输入转文字）、身体状态评分（1-5 星）、老师带课评分（1-5 星）、主题标签（多 chip）、愉悦度 emoji（单选 8 个预设）。系统自动标记记录来源（today_checkin / backfill / csv_import）。

#### Scenario: 下课后快速打卡
- **WHEN** 用户点击底部 + 号或记录页新建按钮
- **THEN** 展示新增表单，日期默认今天，时长默认 60 分钟
- **WHEN** 用户填写机构、老师、课型后保存
- **THEN** 记录保存成功，来源标记为 today_checkin，跳转到记录列表

#### Scenario: 补录历史课时
- **WHEN** 用户修改日期为过去的日期并保存
- **THEN** 记录保存成功，来源标记为 backfill

#### Scenario: 语音输入笔记
- **WHEN** 用户点击笔记区域的麦克风图标并说话
- **THEN** 语音实时转文字填入笔记框，用户可编辑

#### Scenario: 主题标签历史复用
- **WHEN** 用户在标签输入框开始输入
- **THEN** 显示历史使用过的匹配标签供快速选择

### Requirement: 上课记录列表
系统 SHALL 以时间倒序展示用户的上课记录列表，每条显示日期、机构、老师名、课型、时长、首张照片缩略、愉悦度 emoji。支持按机构/老师/月份/课型/标签/愉悦度筛选，支持笔记和标签关键字搜索。

#### Scenario: 查看记录列表
- **WHEN** 用户进入"记录" Tab
- **THEN** 展示全部上课记录，按日期倒序排列

#### Scenario: 多维筛选
- **WHEN** 用户选择筛选条件（如某机构 + 某月份）
- **THEN** 列表仅显示符合条件的记录

#### Scenario: 关键字搜索
- **WHEN** 用户在搜索框输入关键字
- **THEN** 匹配笔记内容或标签的记录被筛出

#### Scenario: 空态展示
- **WHEN** 用户无任何上课记录
- **THEN** 展示空态插画和"去记录第一节课"的引导按钮

### Requirement: 上课记录详情
系统 SHALL 展示单条上课记录的完整信息，包含照片轮播（点击放大）、所有字段详情。支持编辑、删除（二次确认）、一键生成日签（跳转 M5）、以模板新建（快捷复制当前记录的机构/老师/课型/时长创建新记录）。

#### Scenario: 查看记录详情
- **WHEN** 用户在列表中点击某条记录
- **THEN** 进入详情页，展示全部字段和照片轮播

#### Scenario: 编辑记录
- **WHEN** 用户在详情页点击编辑
- **THEN** 跳转到编辑表单，预填当前值，保存后返回详情页刷新

#### Scenario: 删除记录
- **WHEN** 用户在详情页点击删除
- **THEN** 弹出二次确认弹窗
- **WHEN** 用户确认删除
- **THEN** 记录被删除，返回列表页

#### Scenario: 以模板新建
- **WHEN** 用户在详情页点击"以这条为模板新建"
- **THEN** 跳转到新增表单，预填机构/老师/课型/时长，日期重置为今天
